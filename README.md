## 🇧🇷 Português (Brasil)

### ✨ Principais Recursos
- **Aba Karaokê**: Central de busca (nome/código) e gerenciamento de fila principal. Inclui alerta visual (texto piscando) para novos pedidos web.
- **Aba Break**: DJ automático para intervalos, garantindo música ambiente quando a fila estiver vazia.
- **Painel & Controle Web**:
    - **Painel Web**: Exibição de mídias/mensagens em Smart TVs ou PCs via IP local.
    - **Controle Remoto**: Opere o sistema pelo smartphone via QR Code (controle.html).
    - **Aprovação Automática**: Opção de entrada direta na fila para pedidos web.
- **Download Integrado**: Baixe músicas diretamente do YouTube (Vídeo ou MP3) com importação automática.
- **Catálogo Videokê**: Sistema de busca por numeração clássica (Raf Eletronics).
- **Manutenção**: Autoupdate do motor de download (yt-dlp).

## 🇺🇸 English

### ✨ Key Features
- **Karaoke Tab**: Central search (name/code) and main queue management. Features a visual alert (flashing text) for new web requests.
- **Break Tab**: Automatic DJ for intervals, ensuring background music when the queue is empty.
- **Panel & Web Control**:
    - **Web Panel**: Display media/messages on Smart TVs or PCs via local IP.
    - **Remote Control**: Operate the system via smartphone using QR Code (controle.html).
    - **Auto-Approval**: Direct queue entry option for web requests.
- **Integrated Download**: Download songs directly from YouTube (Video or MP3) with automatic import.
- **Classic Catalog**: Search system based on classic numbering (Raf Eletronics style).
- **Maintenance**: yt-dlp download engine autoupdate.
