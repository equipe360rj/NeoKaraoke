System Documentation

1. Karaoke Tab

The system's main entertainment center. In this tab, you can search for songs in your local library by name or code, view the full file list, and manage the playback queue. It allows full player control with Play, Stop, and Restart functions for the current song.

2. Break Tab

Functions as an automatic DJ for intervals. When there are no singers in the queue, the system automatically activates this tab to play background music, ensuring the atmosphere never stays silent. You can set up a specific playlist and adjust the volume independently for the break.

3. Panel Tab

Web Panel

Displays the local IP address that can be accessed by a Smart TV or another PC on the same network. This allows displaying media and content from the Display Control on a secondary screen without cables.

QR Code and Remote Control

The QR Code is the direct link to the controle.html file. It functions as a complete remote control that can be operated via smartphone, allowing you to comma d the system remotely, away from the main PC.

Web Request Control

This section features two fundamental controls:

  • Web Requests: Activates or deactivates searching for new requests from the internet.
  • Auto Approval: When enabled, requests go directly into the Playback Queue without needing manual approval.

Display Control

Contains buttons that activate media elements (videos, images, or backgrounds) being displayed in real-time on the Web Panel.

Classic Videokê Catalog

Search system based on the catalogs of legacy Raf Electronics karaoke machines. Perfect for singers who know their favorite song numbers by heart, making it easier to find the corresponding song title.

4. Download Tab

Integrated with YouTube, it allows you to expand your library instantly. Just type the artist or song name to search for videos, choose the desired format (Video or MP3 Audio), and the system will automatically download and import it to your collection.

5. Help Tab / About

Credits and Information

This section serves as the system's "About" page. Here you can find developer credits and important information regarding the NeoKaraoke licensing.

Search Engine Maintenance

Includes a maintenance button for the download engine (yt-dlp). This feature ensures that the YouTube video search and download system remains up-to-date and fully functional.
